En el HTML ¿Qué son las etiquetas de bloque (block elements)?

Son elementos de html que ocupan todo su contenedor.

En el HTML ¿Qué son las etiquetas de línea (inline elements)?

Son elementos que solo ocupan el ancho necesario para su contenido.

Los elementos de bloque ¿pueden contener en su interior elementos de línea?

Sí.

Los elementos de línea ¿pueden contener en su interior elementos de bloque?

No, no deberían ponerse elementos inline dentro de block elements

Los estilos width y height se aplican a ¿elementos de bloque, de línea o para ambos?

Solo a los en bloque

Los estilos margin y padding se aplican a ¿elementos de bloque, de línea o para ambos?

Se puede a ambos.

¿Qué estilo tienes que aplicar para convertir un elemento de bloque en un elemento de línea?

Para cambiarlo debes poner display:inline.

¿Qué estilo debes aplicar para convertir un elemento de línea en un elemento de bloque?

Para ponerlo como en bloque se pone display:block.

¿Hay algún estilo que convierta a un elemento en ambas cosas a la vez? (es decir, ese elemento será en linea y en bloque a la vez).

Puedes utilizar el display:inline-block.
